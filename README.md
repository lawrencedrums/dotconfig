### Requirements

Install kitty
```
curl -L https://sw.kovidgoyal.net/kitty/installer.sh | sh /dev/stdin
```

Install `neovim`, `ripgreb`, `nerd fonts`

```
brew install neovim
```

```
brew install ripgreb
```

```
brew tap homebrew/cask-fonts
brew install font-hack-nerd-font
```
